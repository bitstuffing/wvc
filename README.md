# Web Video Caster

A lightweight Android app for capturing media links and casting them to Chromecast-compatible devices.

This is an early preview version built to work on devices without Google Play Services or similar dependencies.

## Main features

- Capture multimedia links
- Connect to Chromecast without Google Play Services, GMSCore, or similar components
